Инструкция по настройке VPN

Настройка VPN для Windows (v2rayN)

1. Загрузка v2rayN

Скачайте последнюю версию v2rayN.

2. Установка v2rayN

Распакуйте архив v2rayN.zip и запустите v2rayN.exe из распакованной папки.

Если появится окно с предложением скачать .NET Runtime:

  1. Нажмите "Download it now"
  2. Установите скачанный файл windowsdesktop-runtime-8.0.10-win-x64.exe
  3. Перезапустите v2rayN.exe

3. Настройка языка интерфейса

Смените язык интерфейса на русский или английский:

  1. Нажмите на три точки в верхнем правом углу
  2. Выберите "en" (английский) или "ru" (русский)
  3. Перезапустите приложение

4. Импорт конфигурации VPN

  1. Скопируйте ссылку, предоставленную администратором VPN
  2. В v2rayN выберите: Сервера -> Импорт массива URL из буфера обмена
  3. Обновите подписку: Группа подписки -> Обновить подписку с прокси

5. Установка Xray

  1. Перейдите на страницу Xray-core releases
  2. Скачайте Xray-windows-64.zip
  3. Распакуйте архив и скопируйте wxray.exe и xray.exe в папку v2rayN\bin\Xray

6. Финальная настройка v2rayN

  1. Установите системный прокси
  2. Настройте маршрутизацию
  3. Активируйте режим VPN

Настройка VPN для iPhone/Mac OS

1. Установка приложения

Установите приложение Streisand: https://apps.apple.com/ge/app/streisand/id6450534064

2. Копирование ссылки на подписку

Скопируйте ссылку на подписку, которую вам прислал администратор

3. Добавление подписки

  1. Откройте приложение Streisand
  2. Нажмите "+" в правом верхнем углу
  3. Выберите в появившемся меню пункт "Добавить из буфера"

4. Подключение VPN

  1. Нажмите на иконку подключения
  2. Разрешите добавление профиля VPN
  3. Введите пароль от вашего устройства для подтверждения

5. Проверка срока действия подписки

В нижней части экрана вы увидите количество дней, оставшихся в подписке, а также количество израсходованного трафика.

Настройка VPN для Android

1. Установка приложения

Установите приложение v2rayNG: https://play.google.com/store/apps/details?id=com.v2ray.ang

2. Добавление подключения

  1. Откройте приложение v2rayNG
  2. Нажмите "+" в правом верхнем углу
  3. Выберите в появившемся меню пункт сканирования QR Кода

3. Подключение VPN

  1. Нажмите на иконку подключения
  2. Дождитесь запуска службы VPN

4. Проверка статуса подключения и отключение VPN

В нижней части экрана вы увидите cтатус подключения и кнопку отключения.
