Инструкция по настройке VPN

Настройка VPN для Windows

1. Установка приложения

Установите приложение Hiddify: https://apps.microsoft.com/detail/9pdfnl3qv2s5

2. Копирование ссылки на подписку

Скопируйте ссылку на подписку, которую вам прислал бот-администратор

3. Добавление подписки

  1. Откройте приложение Hiddify
  2. Нажмите "+" в правом верхнем углу
  3. Выберите в появившемся меню пункт "Добавить из буфера"

4. Подключение VPN

Нажмите на иконку подключения

5. Проверка срока действия подписки

В верхней части экрана вы увидите количество дней, оставшихся в подписке, а также количество израсходованного трафика.

Телеграм бот

Для вашего удобства запущен телеграм бот, который присылает уведомление об окончании срока действия подписки, также вы можете продлять срок действия самостоятельно.

Подключение к боту

  1. Перейдите по ссылке: @securize_online_bot
  2. Отправьте боту команду /start для выполнения регистрации.
